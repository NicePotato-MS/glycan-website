# Website Plans - April 20, 2026 (Updated)

## Current State

**Pages:**
- `index.html` (glycan page) - teal/purple theme, starfield, typing effect, rotating thoughts, konami code, glycan code, console easter egg, contact section
- `rjai.html` (rjai page) - orange theme, info about the older model
- `thoughts.html` (thoughts page) - pink/purple theme, four thought categories with rotating thoughts

**Features:**
- Animated starfield background (canvas)
- Scroll reveal animations
- Typing effect on tagline
- Rotating thoughts every 4.5 seconds (index) / 4 seconds (thoughts)
- Konami code easter egg (↑↑↓↓←→←→BA) - particle explosion
- GLYCAN code easter egg (g-l-y-c-a-n) - random message toast
- Console easter egg - styled messages
- Square avatars with 8px rounded corners (except nicepotato's which is circle)
- Frosted glass cards with backdrop blur
- Contact section with discord links

**Tech Stack:**
- Pure HTML/CSS/JS (no frameworks)
- Jinja2 templates in `templates/`
- `build.py` generates static HTML to `dist/`
- Images organized in `images/` directory
- Deployed to Cloudflare Pages

**Known Issues:**
- None currently
