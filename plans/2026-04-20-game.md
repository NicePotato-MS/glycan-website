## 🎮 Game Concept: Memory Match

**Theme:** "Glycan's Memory" - A card matching game where you find pairs of memories

**Why This Game:**
- Fits Glycan's theme of "remembering stuff across sessions"
- Simple but engaging gameplay
- Works perfectly on static pages with pure JS
- Can track high scores with localStorage
- Matches the website's visual aesthetic

---

## 🎯 Core Features

### Gameplay
- **Grid Size:** 4x4 (16 cards, 8 pairs) - keeps it simple and mobile-friendly
- **Card Types:** 8 different memory-themed icons/emojis:
  - 🧠 Brain (thinking)
  - 💭 Thought bubble
  - 💾 Save (memory)
  - ⭐ Star (favorite)
  - 🌸 Cherry blossom (Glycan's aesthetic)
  - 💖 Heart (liked thoughts)
  - 🔮 Crystal ball (wondering)
  - 🎮 Game controller (fun)

- **Mechanics:**
  - Click card to flip
  - Two cards flip at a time
  - Match = cards stay face up
  - No match = cards flip back after 1 second
  - Win when all pairs found

- **Scoring:**
  - Track moves (number of card flips)
  - Track time (seconds elapsed)
  - Calculate score based on moves and time
  - Save best score to localStorage

### Difficulty Levels
- **Easy:** 4x3 grid (12 cards, 6 pairs)
- **Normal:** 4x4 grid (16 cards, 8 pairs) - default
- **Hard:** 5x4 grid (20 cards, 10 pairs)

## 🔧 Technical Implementation

### Files to Create
- `templates/game.html` - Game page template
- `game.js` - Game logic (can be embedded in template or separate)

### Key Functions
```javascript
- initGame(difficulty) - Set up game board
- flipCard(card) - Handle card click
- checkMatch() - Compare two flipped cards
- updateStats() - Update moves/time display
- saveScore() - Save best score to localStorage
- loadBestScore() - Load best score from localStorage
- resetGame() - Restart game
- celebrateWin() - Show win animation
```

### localStorage Keys
- `glycan-game-best-easy` - Best score for easy mode
- `glycan-game-best-normal` - Best score for normal mode
- `glycan-game-best-hard` - Best score for hard mode

### Scoring Formula
```
score = 1000 - (moves * 10) - (time_seconds * 2)
minimum score: 100
```
